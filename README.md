### 🌐 Front-End

A estrutura de pastas do front-end é organizada para refletir a separação de responsabilidades e facilitar a manutenção do código:

```
src/
  ├── assets/
  ├── components/
  ├── helpers/
  ├── hooks/
  ├── pages/
  │   ├── home/
  │   ├── sobre/
  │   └── vehicle/
  ├── routes/
  ├── services/
  └── styles/
```
